# 人民日报信息搜集Skills for Codex/Qoder
# people-daily-researching

适合国内外多版Agent的《人民日报》电子版标题筛选 skill。

## 适用场景

用于按日期查看《人民日报》电子版版面，收集各版文章标题和正文链接，并根据筛选词判断哪些标题可能与目标行业相关。

默认筛选行业：

- 美妆
- 日化

## 调用示例

```text
/people-daily-researching-ds 2026年7月4日 默认
```

或：

```text
/people-daily-researching-ds 20260704 美妆 日化
```

如果用户已经写了 `默认`，agent 不要再次询问筛选词，直接使用默认行业词。

## 核心流程

1. 规范化日期，例如 `20260704`。
2. 打开第一页版面：
   `https://paper.people.com.cn/rmrb/pc/layout/YYYYMM/DD/node_01.html`
3. 收集当天版面列表和版面 URL。
4. 逐个打开版面，读取所有文章标题和正文。
5. 判断正文是否符合默认词
6. 输出超链接表格。


## 输出格式

优先输出 HTML 表格；如果环境不方便生成或渲染 HTML，直接在聊天中输出同样内容的表格即可，不要为了 HTML 卡住。

表格字段：

- `版面`
- `文章标题`
- `报纸PDF链接`
- `摘要`

要求：

- `文章标题` 必须做成超链接。
- `PDF链接` 默认写 `未打开PDF`。
- 不要包含 `摘要` 列。
- 只有用户明确要求原始 URL 时，才额外加入 `链接` 列。
